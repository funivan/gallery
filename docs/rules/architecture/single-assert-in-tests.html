Architecture :: Single assert in tests Recommendation! This can be a good practice to use only one assertion in the Unit Tests 1. Bad public function testUrlBuild(){ $url = new Url('userName') self::assertSame('/url/userName', $url->build()) self::assertCount(0, $url->parameters()) } 2. Good public function testUrlBuildToString(){ $url = new Url('userName') self::assertSame('/url/userName', $url->build()) } public function testUrlParameters(){ $url = new Url('userName') self::assertCount(0, $url->parameters()) }
